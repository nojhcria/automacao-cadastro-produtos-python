# Automação de Cadastro de Produtos com Python

## 📌 Sobre o projeto

Este projeto foi desenvolvido para automatizar o cadastro de produtos em um sistema web utilizando Python.

A aplicação realiza a leitura de uma base de dados em formato CSV com a biblioteca Pandas e utiliza o PyAutoGUI para preencher automaticamente os campos do sistema, reduzindo tarefas repetitivas e aumentando a produtividade.

## 🚀 Tecnologias utilizadas

- Python
- Pandas
- PyAutoGUI

## 📂 Estrutura do projeto

```
automacao-cadastro-produtos-python/
│
├── main.py
├── produtos.csv
├── requirements.txt
└── README.md
```

## ⚙️ Funcionalidades

- Leitura da base de produtos (CSV)
- Automação do login no sistema
- Cadastro automático de produtos
- Tratamento de dados com Pandas
- Automação de tarefas com PyAutoGUI

## 🎯 Objetivo

Praticar automação de processos utilizando Python e aplicar manipulação de dados para otimizar tarefas operacionais.
